# FreeLum

Free, open-source screen recorder for macOS with webcam overlay. A lightweight alternative to Loom.

## Features

- Screen recording with H.264 encoding
- Webcam overlay (circle, rounded rectangle, or pill shape)
- Microphone audio capture
- 3-2-1 countdown before recording
- Pause / resume support
- Global hotkeys: `⌘⇧R` record, `⌘⇧P` pause
- Drag to move webcam, scroll to resize
- Snap webcam to any screen corner
- Lives in the menu bar — no dock icon

## Requirements

- macOS 14.0+
- Xcode 15+

## Build

```bash
git clone https://github.com/vibecoderkz/freelum.git
cd freelum
open loom.xcodeproj
```

Build and run in Xcode. Grant Screen Recording, Camera, and Microphone permissions when prompted.
